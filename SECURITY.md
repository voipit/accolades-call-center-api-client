# Security Policy

## Supported Versions

Security updates are provided only for the latest released version of this connector.

Users are encouraged to keep their deployments up to date.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them privately by contacting the repository maintainer.

When reporting a vulnerability, please include:

* A description of the issue.
* Steps required to reproduce the issue.
* The connector version affected.
* The potential impact of the vulnerability.
* Any suggested remediation, if available.

Reports will be reviewed and addressed as promptly as possible.
